## 强烈推荐：Claude

claude 的 pdf 理解能力非常强（T0），是图文混合多模态，非常推荐优先尝试。

![img.png](../../assets/claude-multimodel.png)

但它的 API 端限制较大，并发不满足大规模需求。

![img.png](../../assets/claude-api-rate-limit.png)

前端逆向的 API 或许可以尝试一下，因为可以传的文件更多，rate-limit 也稍微宽松一些，或者找专门的号池。
